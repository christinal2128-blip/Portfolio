# Portfolio Code References

This file lists beginner-friendly documentation for the main ideas used in this portfolio.

## HTML structure

- `nav` for the navigation links:
  https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/nav
- `main` for the main page content:
  https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/main
- `aside` for side content such as the homepage focus box:
  https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/aside
- `footer` for the bottom section of each page:
  https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/footer

## CSS layout and styling

- CSS custom properties (`--color-name`) for theme colors:
  https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Cascading_variables/Using_custom_properties
- Flexbox for header and button alignment:
  https://developer.mozilla.org/docs/Web/CSS/CSS_Flexible_Box_Layout/Basic_Concepts_of_Flexbox
- CSS Grid for the hero, project cards, and two-column layouts:
  https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_grid_layout
- Media queries for responsive design on smaller screens:
  https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_media_queries/Using_media_queries

## JavaScript behavior

- `localStorage` for saving the dark/light theme choice:
  https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage
- `data-*` attributes, which are similar to the `data-theme` value used on the `body`:
  https://developer.mozilla.org/docs/Learn/HTML/Howto/Use_data_attributes

## Accessibility

- `aria-label` for giving the icon-only theme button an accessible name:
  https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-label
